# Document AI Lab

OCR, preprocessing, and document workflow experiments.

## Purpose

This repository collects practical experiments around scanned documents, image preprocessing, OCR quality, text extraction, and post-processing workflows.

## Planned Experiments

- image cleanup before OCR;
- PDF page rendering;
- text extraction and normalization;
- OCR result comparison;
- field extraction from receipts, forms, and technical documents;
- evaluation notes for failure cases.

## Tech Stack

- Python
- OpenCV / Pillow
- OCR engines and APIs
- CLI scripts
- notebooks or small reproducible examples

## Repository Structure

```text
examples/      sample inputs and outputs
scripts/       reusable CLI tools
notebooks/     experiments and analysis
docs/          workflow notes and diagrams
```
